MMT archives correspond to software projects and provide project-oriented work flow for any language defined in MMT.
They are the primary means to maintain and distribute MMT content.

A number of MMT archives are described here.

Writing Archives

To write an MMT archive, check out https://tntbase.mathweb.org/repos/oaff/empty-archive to obtain an empty example archive to play with and adapt.

Archives can be written in any syntax that MMT can understand natively or import.
The MMT IDE based on jEdit can be used well to write archives in MMT's native text syntax.
The MMT build tool can be used to build MMT archives. This is necessary to make the content available to othe MMT applications.
