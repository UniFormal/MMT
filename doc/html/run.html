After checking out and optionally building MMT, the jars in the deploy/ folder an be run directly.
However, because MMT is language and application-independent, MMT by itself does not do anything - it must be customized by defining a language in the MMT language and building an application on top of the MMT system.
Of course, the MMT system suite includes various language definitions and applications to use.
See the respective pages for instructions on loading individual languages and running individual applications.
