MMT is used in the LATIN project. In particular, an instance of the LATIN web browser can be found here.
In particular, LATIN uses MMT notations to produce JOBAD-enabled XHTML+presentation MathML.

The Hets system (written in Haskell) uses MMT to generate implementations of its abstract logic class from declarative logic definitions in MMT/LF.

MMT and JOBAD were used in our submission to the AI Mashup Challenge.

Information on the MMT-Plugin for TNTBase can be found here.

The MMT language is used as the basis in the Twelf module system. See here for more information.
