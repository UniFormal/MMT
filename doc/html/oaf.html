The OAF is a repository backend for mathematical content hosted at gl.mathhub.info.
It is the main place to develop and publish MMT archives.

Archives

In particular, the OAF contains the following MMT archives:
archivecontentsdepends on
MMT/urtheoriesvarious top level languages, in particular the logical framework LF
MMT/examplesan example archive for beginnersMMT/urtheories
MMT/experimentalsimilar to MMT/examples but not always workingMMT/urtheories
MMT/LATINthe LATIN logic atlasMMT/urtheories
Mizar/mmlthe Mizar Mathematical LibraryMMT/LATIN
HOLLight/basicthe standard library of HOL LightMMT/LATIN

Setup

To work with the archives in the OAF, you usually want to clone the relevant git repositories and then add your local working copies to the math path.
To clone the repositories, you typically proceed along the following lines: 
mkdir archives
cd archives

mkdir MMT
cd MMT
git clone git@gl.mathhub.info:MMT/urtheories.git
git clone git@gl.mathhub.info:MMT/examples.git
git clone git@gl.mathhub.info:MMT/LATIN.git
cd ..

mkdir HOLLight
cd HOLLight
git clone git@gl.mathhub.info:HOLLight/basic.git
cd ..

mkdir Mizar
cd Mizar
git clone git@gl.mathhub.info:Mizar/MML.git
cd ..
