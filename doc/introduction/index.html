This directory holds introductory documents about MMT:
  • mmt.pdf: a general overview of MMT
  • mmt-slides.pdf: introductory slides on MMT
  • mmt-vs-X.pdf: a list of related systems together with a short explanation how they relate to MMT
LaTeX sources are in the same directory, and you may use any parts of these documents if you want to give a presentation about MMT.
