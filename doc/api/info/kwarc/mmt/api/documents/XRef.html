XRef

Related Doc: package documents

abstract class XRef extends NarrativeElement

An XRef represents a reference from a document to an external document part or module. An XRef is semantically equivalent to copy-pasting the referenced module. All documents are reprgesented as lists of XRefs no matter whether a module is given locally or remotely.
