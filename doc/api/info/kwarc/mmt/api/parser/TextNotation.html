TextNotation

case class TextNotation (markers: List[NotationElement], precedence: Int, isBinder: Boolean, conPath: GlobalName) extends Notation with Product with Serializable

A notation is a list of markers which represent either strings to be matched or argument positions to be filled. It also has notation properties encoding things such as associativity or precedence. e.g. given the binary operator log(nr,base) one can define the notation with markers List("log",2,1) where "log" is a string marker, 2 is a argument marker for argument number 2 of log (base) and 1 for argument number 1 (nr). Then, log 2 16 is parsed as log(16,2)
