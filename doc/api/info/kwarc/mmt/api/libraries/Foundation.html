Foundation

class Foundation extends AnyRef

MMT Foundation: provides oracles for typing and equality. Concrete foundations are registered as plugins and maintained by @frontend.ExtensionManager Concrete foundations must have a constructor that takes no arguments, which will be called after plugin registration
