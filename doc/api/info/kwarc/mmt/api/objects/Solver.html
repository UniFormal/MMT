Solver

class Solver extends AnyRef

A Solver is used to solve a system of constraints given as judgments about Term's by applying typing rules to validate the judgments. The judgments may contain unknown variables (also called meta-variables or logic variables); variables may represent any MMT term, i.e., object language terms, types, etc.; the solution is a Substitution that provides a closed Term for every unknown variable. (Higher-order abstract syntax should be used for problem where the solutions are not closed.) Unsolvable constraints are delayed and reactivated if solving other constraints provides further information.
