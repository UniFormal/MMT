RuleStore

class RuleStore extends AnyRef

A RuleStore maintains sets of foundation-dependent rules that are used by a Solver.

It maintains a separate set of TypingRule's for every subtype of Rule; the Rule's are indexed by their heads. An instance of RuleStore is created by the ExtensionManager.
