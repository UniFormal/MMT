DeclaredTheory

class DeclaredTheory extends Theory with DeclaredModule[Symbol]

A Theory represents an MMT theory.

Theories are constructed empty. info.kwarc.mmt.api.modules.StatementSet[Symbol] is derived to hold a set of named symbols.
