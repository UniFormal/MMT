DeclaredView

Related Doc: package modules

class DeclaredView extends View with DeclaredModule with DeclaredLink

A DeclaredView represents an MMT view given by a list of assignments.

Assignments are Constants, whose name is the qualified name (always starts with ComplexStep) of a domain Constant and whose definiens is codomain Term.
