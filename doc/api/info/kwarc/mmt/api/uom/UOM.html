  29. def simplify(t: Term, scope: Term, context: Context = Context()): Term

    the main simplification method

    the main simplification method

    t

    the term to simplify

    context

    its context, if non-empty

    returns

    the simplified Term (if a sensible collection of rules is used that make this method terminate)

    The input term must be fully strictified, and so will be the output term. Applicability of rules is determined based on the pragmatic form (using controller.pragmatic.StrictOMA). Rules are passed strict terms and are expected to return strict terms.

    The code uses Simple to remember whether a term has been simplified. So multiple calls to this method do not cause multiple traversals.
