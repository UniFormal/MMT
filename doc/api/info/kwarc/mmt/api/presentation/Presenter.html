Presenter

Related Docs: object Presenter | package presentation

abstract class Presenter extends BuildTarget with Exporter with StructurePresenter with ObjectPresenter with LeveledExtension

the designated super class of all presenters

The format for which a Presenter is applicable is the same as the key used to run it as an exporter.
