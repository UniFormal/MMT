class Facts extends Logger

A database of facts obtained through forward proof search

For efficiency, each instance only searches for terms that are added when the context is enriched. Therefore, each Goal g maintains one instance of Facts, which links to the instance of the g.parent. Each instance knows the local context of its goal, and maintains only terms that use a local variable.
