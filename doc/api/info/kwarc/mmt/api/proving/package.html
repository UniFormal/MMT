  13. class Goal extends AnyRef

    a single-conclusion sequent - the basic node in a proof tree

    a single-conclusion sequent - the basic node in a proof tree

    The root goal is stored and the whole proof tree is acted on by the Prover.

    A goal nodes knows its parent (except for the root goal) and children (the subgoals). In fact, a goal is also a node in the backwards proof search: A goal stores not simply a list of subgoals, but a list of Alternative ways to prove it each of which stores a list of subgoals.

    Moreover, a goal stores the explored part of a the forward proof search space: a set of facts implied by the goals premises, updated externally.

    The prover expands new goals greedily by applying invertible rules, and each goal stores those invertible rules that have not been applied yet.
