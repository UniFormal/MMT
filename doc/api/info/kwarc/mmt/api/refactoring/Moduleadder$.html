Moduleadder

Related Doc: package refactoring

object Moduleadder

Takes a View/Theory T and a (Set of) (Pair(s) of) Constant(s) (either as FinalConstant of GlobalName) and adds them to T consistent with the specifications (i.e. adapts the GlobalName accordingly and so on). Also, if handed a theory and a Set of Constants, recursively substitutes occurences of the other members of the set in types/definitions
