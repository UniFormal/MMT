trait Body[S <: Declaration] extends AnyRef

StatementSet represents the content of modules, i.e., a set of declarations.

It provides (i) a name-indexed hash map of symbol level Declarations, and a list of Imports. It is mixed into theories and links to hold the symbols/imports and assignments/imports, respectively.

known subclasses: DeclaredModule, DeclaredLink
